Strings surrounded by either double quotation or single quotation marks

Double quoted strings returns value of variable
example = 
            $x = "John";
            echo "Hello $x";

Single quoted strings returns string as it was written with variable name

example = 
            $x = "John";
            echo 'Hello $x';
